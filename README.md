# steamscript
Depends on [python-valve](https://github.com/Holiverh/python-valve)

How to use pythonpath to find python-valve
```
cd GITDIR
git clone https://github.com/Holiverh/python-valve

cd -
PYTHONPATH=$GITDIR python rusts.py
```
